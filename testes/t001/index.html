Para fazer uma quebra de texto, basta usar a tag <br>
assim.

Para usar caracteres especiais e símbolos, existe uma tabela no PDF do curso, ou basta procurar por HTML Entities no Google.
Alguns exemplos: < > ™ © ® & " ' ¢ £ € ¥  

Para usar emojis, basta pesquisar pelos códigos em emojipedia.com, e sempre usar o código "&#x" e o código específico do emoji na frente.
Alguns exemplos: 😄 ⚽ 🇧 🇷
